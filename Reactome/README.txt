The mapping files consist of a tab-separated table that indicates which external protein, gene or small molecule identifiers in the source database were mapped to Reactome pathway and reaction annotations.

The columns within the mappings files follow a similar format:

1 Source database identifier, e.g. UniProt, ENSEMBL, NCBI Gene or ChEBI identifier
2 Reactome Pathway Stable identifier
3 URL
4 Event (Pathway or Reaction) Name
5 Evidence Code
6 Species
